# Структура папок для медиа-ресурсов

Эта папка содержит все медиа-файлы для VideoLab.

## Структура:

- **Video/** - Видео файлы
  - backgrounds/ - Фоновые видео
  - effects/ - Видео эффекты
  - transitions/ - Переходы между сценами
  - footage/ - Футажи

- **Audio/** - Аудио файлы
  - music/ - Фоновая музыка
  - sounds/ - Звуковые эффекты

- **Picture/** - Изображения
  - backgrounds/ - Фоновые изображения
  - stickers/ - Стикеры и эмодзи
  - icons/ - Иконки и значки
  - previews/ - Превью изображения

- **Fonts/** - Шрифты
  - Файлы шрифтов
